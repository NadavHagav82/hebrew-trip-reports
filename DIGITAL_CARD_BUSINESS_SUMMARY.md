## 💰 מודל הכנסות

### מקור הכנסה עיקרי: Interchange Fee

```
משתמש משלם $100 במסעדה
  → הסוחר משלם ~2% עמלה ($2)
    → חלוקה:
      ├── אנחנו (המנפיק): ~$1.00 (1%)    ← ההכנסה שלנו
      ├── Stripe:            ~$0.40 (0.4%)
      └── Visa/MC:           ~$0.60 (0.6%)
```

### אין עלות למשתמש על עסקאות:
- ❌ אין דמי הקמה
- ❌ אין דמי שימוש חודשי
- ❌ אין עמלה על עסקאות
- ✅ הכל חינם — הסוחר משלם

## 🔄 טעינת ארנק — עמלות לפי שיטה ומדינה

### 🇺🇸 ארה"ב:

| שיטה | עמלה | זמן | הערה |
|---|---|---|---|
| **ACH (בנק)** | **$0.25 קבוע** | 1-2 ימים | ✅ מומלץ — כולם מכירים |
| **Stripe Treasury** | **$0** | מיידי-יום | ✅ הכי זול |
| **כרטיס דביט** | **1.5%** | מיידי | סביר |
| **כרטיס אשראי** | **2.9% + $0.30** | מיידי | יקר, לחירום |

**ACH = כמו ביט בישראל.** מחברים בנק פעם אחת דרך Plaid → אחרי זה טעינה בלחיצה אחת.

### 🇪🇺 אירופה:

|---|---|---|
| **SEPA** | **~€0.35 קבוע** | 1-2 ימים |
| **כרטיס אשראי** | **2.9%** | מיידי |

### 🇬🇧 בריטניה:

### 🇮🇱 ישראל:

| שיטה | עמלה | זמן | הערה |
|---|---|---|---|
| **כרטיס אשראי** | **2.9%** | מיידי | ✅ פתרון יחיד פרקטי |
| **Apple/Google Pay** | **2.9%** | מיידי | ✅ חוויה טובה יותר |
| **העברה בנקאית** | **~$30 קבוע** | 1-3 ימים | משתלם רק מ-$1,000+ |

#### נקודת איזון בישראל:
```
מתחת ל-$1,000 → כרטיס אשראי (2.9%) זול יותר מהעברה ($30)
מעל $1,000   → העברה בנקאית זולה יותר
$1,034       → נקודת האיזון בדיוק
```

## 🧮 כלכלת עסקה — דוגמאות

### עצמאי ישראלי (טעינה $1,000):
```
טעינה: $1,000 → גובים $1,029 (המשתמש משלם 2.9%)
הוצאות: $1,000 בנסיעה
Interchange: ~$10 (1%)
עלות טעינה לנו: $0 (המשתמש שילם)
רווח נקי: $10 ✅
```

### ארגון אמריקאי (טעינה $10,000 ACH):
```
טעינה: $10,000 → עמלה $0.25
הוצאות: $10,000 בנסיעות
Interchange: ~$100 (1%)
רווח נקי: $99.75 ✅✅
```

### ארגון ישראלי (העברה בנקאית $20,000):
```
טעינה: $20,000 → עמלת בנק ~$30 (0.15%)
הוצאות: $20,000 בנסיעות
Interchange: ~$200 (1%)
רווח נקי: $200 ✅✅✅
```

## 📱 חוויית משתמש

### טעינה (ACH — ארה"ב):
```
פעם ראשונה: "Link Bank" → Plaid → בוחר בנק → מתחבר → ✅ (30 שניות)
כל פעם אחרי: "Fund $1,000" → ✅ לחיצה אחת
```

### טעינה (כרטיס אשראי — ישראל):
```
"טען ארנק" → $1,000 → Apple Pay / פרטי כרטיס → ✅ מיידי
(מוצג: "סה"כ לתשלום: $1,029 כולל עמלת סליקה")
```

### תשלום בנסיעה:
```
מוציא טלפון → Apple/Google Pay → מקרב לקורא → ✅
  → התראה מיידית: "חויבת $85 במסעדה"
  → הוצאה נרשמת אוטומטית
  → צלם חשבונית ← AI מוודא התאמה
```

### משיכת יתרה:
```
יתרה לא מנוצלת → "משוך כסף" → חוזר למקור ההפקדה בלבד (AML)
```
