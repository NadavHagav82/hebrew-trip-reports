**השורה התחתונה:**

- 📈 הכנסה שגדלת לינארית עם כל לקוח חדש

## 📊 מהיכן מגיעה ההכנסה?

### מודל Interchange Revenue

כשעובד משלם בכרטיס שלנו, **הסוחר** (מלון, מסעדה, חברת תעופה) משלם עמלה:

```
עובד משלם 1,000$ במלון
  → המלון משלם ~2% עמלה (20$)
    → Visa/Mastercard: ~0.5% (5$)
    → Stripe: ~0.5% (5$)
    → אנחנו: ~1% (10$) ← ההכנסה שלנו
```

### למה זה מודל עסקי מעולה:
| מאפיין | הסבר |
|---------|--------|
| 🔄 חוזר | כל חודש, כל נסיעה |
| 📈 גדל | יותר לקוחות = יותר הכנסה |
| 💤 פסיבי | ברגע שלקוח מחובר — הכסף זורם |
| 🆓 ללא חיכוך | הלקוח לא משלם — אז לא מתלונן |
| 🏦 מוכח | זה המודל של Brex, Ramp, Navan — חברות מיליארד דולר |

## 🔒 Lock-in ושימור לקוחות

### היום (בלי כרטיס):
```
לקוח רוצה לעזוב ← מייצא נתונים ← עובר למתחרה ← קל
```

### עם כרטיס דיגיטלי:
```
לקוח רוצה לעזוב ← צריך לבטל כרטיסים לכל העובדים
                  ← צריך להנפיק כרטיסים חדשים אצל מתחרה
                  ← צריך להעביר תקציבים
                  ← צריך להגדיר מדיניות מחדש
                  ← כאב ראש ← נשאר ✅
```

**שיעור נטישה צפוי:** יורד מ-15-20% ל-**3-5%** עם כרטיס דיגיטלי.

## 🎯 יתרונות אסטרטגיים (מעבר לכסף)

### 1. 🏷️ מיצוב בשוק
```
היום: "עוד מערכת דיווח הוצאות" ← שוק צפוף
מחר: "פלטפורמה פיננסית לנסיעות עסקיות" ← שוק פרימיום
```

### 2. 📊 נתונים = זהב
כרטיס דיגיטלי נותן לנו **נתוני הוצאות אמיתיים** ברזולוציה שאף מתחרה לא יכול להשיג:
- איפה עובדים אוכלים
- כמה עולה מלון ממוצע בכל עיר
- מגמות הוצאות לפי עונה
- **← מוצרי אנליטיקות פרימיום בעתיד**

### 3. 🤝 שותפויות
עם מחזור עסקאות משמעותי:
- חברות תעופה מציעות עסקאות
- רשתות מלונות רוצות להיות "ספק מועדף"
- חברות ביטוח מציעות שילוב
- **← הכנסות נוספות משותפויות**

### 4. 🚀 הערכת שווי (Valuation)
| מודל הכנסה | מכפיל הערכה |
|------------|-------------|
| SaaS בלבד | ×5-8 הכנסות שנתיות |
| SaaS + Interchange | **×10-15** הכנסות שנתיות |
| Fintech | **×15-25** הכנסות שנתיות |

```
שנה 3 הכנסות: $192,000
× מכפיל SaaS (×7): שווי $1.3M
× מכפיל Fintech (×15): שווי $2.9M ← כמעט פי 2!
```
